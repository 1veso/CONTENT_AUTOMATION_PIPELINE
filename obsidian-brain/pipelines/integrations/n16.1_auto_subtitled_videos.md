# n16.1 — Auto-Subtitled Videos (ElevenLabs → NCA Toolkit)

## What it does
Companion flow to [[n16_narrative_chaining]]. Takes a generated video + ElevenLabs voiceover and burns synchronized subtitles via the NCA Toolkit (whisper-style alignment + style overlay).

## Tools used
- ElevenLabs (voice)
- NCA Toolkit (transcribe + burn subtitles)
- Cloudinary or Box for media handoff

## Credentials needed
- `ELEVENLABS_API_KEY`
- `NCA_TOOLKIT_*` (endpoint + auth)
- `CLOUDINARY_URL` / `BOX_*`
